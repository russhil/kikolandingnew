# Kiko AI Order Taker — Landing Pages

Crisp, mobile-first landing pages built to pair with Meta ads (Gujarat-first,
wholesalers/distributors). Designed for **~5–6 seconds of attention**: one
headline, one animated proof, one action — **Install the app**.

## Structure

```
/            → language chooser (links to the 4 versions below)
/en/         → English
/hi/         → हिन्दी (Hindi)
/gu/         → ગુજરાતી (Gujarati)
/mr/         → मराठी (Marathi)
```

Each language folder is **fully self-contained** (`index.html` + `logo_kiko.png`),
so any one can be deployed independently. Pages are pure HTML + inline CSS +
~15 lines of inline JS — **no build step, no external CDNs** — so they open
instantly on a 4G phone and work on both mobile and desktop (two-column hero on
wide screens, single column on mobile).

## Download CTA

Every "Install" button points to the published app:
`https://play.google.com/store/apps/details?id=com.kikocall`

It's set **once** per page via the `PLAY_STORE_LINK` constant in the script at the
bottom of each `index.html`. Append `?utm_source=meta&utm_campaign=...` there and
all CTAs on that page update together.

## Deploy (GitHub Pages)

Settings → Pages → deploy from `main` / root. The chooser is served at the site
root and each language at `/en/`, `/hi/`, `/gu/`, `/mr/`. Point your ads straight
at a language folder (e.g. `…/gu/`) for the Gujarat campaign.

## Editing copy

English (`en/index.html`) is the source of truth. Every translatable line is
tagged with an `i18n` comment. The translated files were generated by
swapping those tagged strings — layout, colours and the hero animation are
identical across all four languages.

## Design notes

- Palette matches the **logo**: indigo `#4324a6` + cyan `#06b6d4`; green
  `#16a34a` is reserved for the Install button so the action always stands out.
- Hero shows a looping CSS animation: **Call → Kiko → Order card → WhatsApp**
  (no video). Respects `prefers-reduced-motion`.
- "As Seen on Shark Tank India S3" trust badge in the top bar and final CTA.
